Output (輸出)

基本輸出

程式錯誤種類

            
            """
            # 題意：將 a 和 b 以算術加法合計

            1. 語法錯誤 Syntax Error  : 不可執行
            2. 執行錯誤 Runtime Error : 可以執行，過程出錯
            3. 邏輯錯誤 Logic Error   : 可以執行，不會出錯，結果不符預期

            type(data) 檢查資料的類別

            str(123)   將整數轉成數字 -> string

            a = input('Input a number: ))

            print('type of a:', type(a))
            print('type of b:', type(b))
            print(a + b)

Repetition (重複) - for loop 迴圈式(迴圈或重複結構)

輸出指定符號

            
            for i in range(5):
                print('*')

輸出指定數列

            
            for i in range(5):
                print(i)

迴圈與決策

            
            """
            # ex(+). diff-stars
            # 不同星號數
            # i 等於 3 時，輸出 ***，其他值則輸出 *

            1 *
            2 *
            3 ***
            4 *
            5 *
            """

            # range(stop): 0 ~ stop-1
            # range(start, stop): start ~ stop-1
            for i in range(1, 6):
                if False:
                    print(i, '***')
                else:
                    print(i, '*')

            """
            # ex(+). triangles

            1 *****

            3 ***
            4 **

Repetition (重複) - while loop 迴圈式(迴圈或重複結構)

while 布林變數

            
            isplus = True
            total = 0
            while isplus:
                total = total + 1
                print(total)
                if total == 10:
                    isplus = False

while 布林值

            
            total = 0
            while True:
                total += 1
                print(total)
                if total == 10:
                    break

等差數列

            
            """
            # ex(+). arithmetic-seq
            # 等差數列
            # 維持使用一個條件式
            # 比較雙方維持 total 和 10

            # 比較運算子
            == 等於     != 不等於
            >= 大於等於  > 大於
            <= 小於等於  < 小於

            [輸入與輸出]
            step: 2
            2
            4
            6
            8
            10

            step: 3
            3
            6
            9

            step: 4
            4
            8
            """

            total = 0
            while True:
                total += 1
                print(total)
                if total == 10:
                    break

while 條件式

            
            """
            # ex(+). while-condition
            # 條件迴圈式
            # while 依條件式決定是否執行
            # 程式行數必須減至 4 行
            """

            total = 0
            while False:
                total += 1
                print(total)
                if total == 10:
                    break
