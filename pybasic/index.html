Output(輸出)

基本輸出

            
            """
            # 內建函式 print()
            # 功能：輸出訊息、變數內容或資料值
            # print(參數：變數或資料值)

            設定函式呼叫步驟：
            1. 設定函式名稱
            2. 放一對小括 ()
            3. 放參數
            """

            # print(1個參數)
            # 輸出一個字串
            print('Hello, World!')

            # print(0個參數)
            # 單純換行
            print()

            # print(2個以上參數，逗號分隔)
            # 參數輸出時，彼此之間空一白
            print('Hello, World!', 'I am R987.')

            # 可以輸出中文或多國語言文字
            print('哈囉！我是 R987。')

輸出單行字串

            
            """
            可用單引號或雙引號設定字串
            必須成對
            """

            # 單引號 (字串單引號 vs. 英文單引號)
            print('What\'s going on?')
            # 雙引號
            print("What's going on?")

輸出不同類別

            
            """
            # 內建函式 type()
            # 功能：回傳變數或資料值的類別
            # type(參數：變數或資料值)
            """

            # 字串 str (string)
            print('123', type('123'))

            # 整數 int (integer)
            print(123, type(123))

            # 浮點數 float
            print(123.0, type(123.0))

            # 布林數 bool (boolean)
            # 例一：硬幣的正反面
            # 例二：電源的開和關
            print(True, type(True))
            print(False, type(False))

字串格式化

            
            """
            字串格式化 (String Formatting)
            1. 小數格式：保留小數位數
            2. 對齊格式：(略，以後再講)

            f-string 設定步驟：
            1. 設定小寫 f
            2. 接一對字串單引號 '' 或雙引號 ""
            3. 視需要放字串常值
            4. 變數或運算式放在一對大括 {} 中
            5. 特定格式要在變數或運算式後接英文冒號 : 再設定
            6. 小數位數例如 .2f 保留小數後 2 位
            """

            # |N = | 是字串常值
            # |n| 是變數
            n = 3
            print(f'N = {n}')

            # |PI = | 是字串常值
            # |pi| 是變數
            # |.4f| 是設定保留小數後 4 位
            pi = 3.1415926
            print(f'PI = {pi:.4f}')

七種算術運算

            
            # 七種算術運算
            """
            # ex(+).
            # 輸出算術運算結果
            """

            # 加(+)、減(-)、乘(*)、除(/)
            7 + 2
            7 - 2
            7 * 2
            7 / 2
            # 商數(//)、餘數(%)
            # % 其實是模數運算
            7 // 2
            7 % 2
            # 指數 (次方)
            7 ** 2

運算子優先序

            
            # 運算子優先序
            """
            # ex(+).
            # 小括 > 指數 > 乘除、商餘 > 加減
            # 先乘除後加減
            # 指乘除餘加減

            # 整數和浮點數計算結果，會得到浮點數
            例如 5 + 1.0 -> 6.0

            Python 的 3 種除法：
            1. /  : 一般除法，得到浮點數
            2. // : 整數相除，取商數
            3. %  : 整數相除，取餘數
            """

            a = 7 + 2 - 3 * 2
            print('a:', a)

            b = 8 + 4 / 2 - 5 * 2
            print('b:', b)

            c = 8 * 2 + 7 - 9 / 4 * 3 ** 2
            print('c:', c)

            d = (8 + -4) / 2 - 5 * 2
            print('d:', d)

            e = 27 % 5 * 6 ** 2 - 12 // 6
            print('e:', e)

            type(data): 回傳資料的類別
            int('987'): 將數字字串轉成整數 -> integer
            str(12345): 將整數轉成數字字串 -> string

            b = (input(b: ')

            print(f'b = {b}', type(b))
            print(a + b)

串列操作

一行三數

            
            # 一行三數
            """
            str.split(): 字串的切割方法
            # str 指某個字串物件
            # split() 依空白切割，將切割後的小字串存到 list(串列)

            alist: 通常表示一個串列
            # alist[index]: 依編號取值，中括 [] 內的整數 index 表示資料編號
            # index 從 0 開始，即 0 為第一個，而 1 為第二個，依此類推

            [輸入]
            2 3 5
            [輸出]
            4 9 25
            """

            alist = input().split()
            print(alist)
            alist[0] = int(alist[0])
            alist[1] = int(alist[1])
            alist[2] = int(alist[2])
            print(alist)
            alist[0] = alist[0] ** 2
            alist[1] = alist[1] ** 2
            alist[2] = alist[2] ** 2
            print(alist)
            # 展開 alist 中的資料
            print(*alist)

一行多數

            
            # 一行多數
            """
            # 第一行有一個整數 N 表示第二行有 N 個整數
            # 第二行即 N 個整數，以空白分隔

            [輸入]
            5
            2 3 5 7 11
            [輸出]
            4 9 25 49 121
            """

            n = int(input())
            alist = input().split()
            for i in range(n):
                alist[i] = int(alist[i]) ** 2
            print(*alist)

存入不定個數

            
            # 存入不定個數
            """
            # 存入不定個正整數，如果輸入值是 -1 表示結束
            # 不要將 -1 存入串列

            [輸入]
            2
            3
            5
            7
            11
            -1
            [輸出]
            2 3 5 7 11
            """

            # 必須在迴圈前設定一個空串列，準備放入資料
            alist = []
            while True:
                i = int(input())
                if i == -1:
                    break
                # list.append(item): 串列的新增方法
                # 在串列物件 alist 後設定 append 方法
                # 將要存入串列的項目，放到 append 方法的小括 () 中
                alist.append(i)
            # 展開並輸出串列所有項目
            print(*alist)
