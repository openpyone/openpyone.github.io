Output (輸出)

基本輸出

            
            """
            # 內建函式 print()
            # 輸出訊息、變數內容或資料值
            # print(參數：變數或資料值)
            """

            # print(1個參數)
            print('Hello, World!')
            # print(0個參數)
            print()
            # print(2個以上參數，逗號分隔)
            print('Hello, World!', 'I am R987.')
            print('哈囉！我是 R987。')

輸出單行字串

            
            """
            可用單引號或雙引號設定字串
            必須成對
            """

            # 單引號 (字串單引號 vs. 英文單引號)
            print('What\'s going on?')
            # 雙引號
            print("What's going on?")

輸出不同類別

            
            """
            # 內建函式 type()
            # 回傳變數或資料值的類別
            # type(參數：變數或資料值)
            """

            # 字串 str (string)
            print('123', type('123'))
            # 整數 int (integer)
            print(123, type(123))
            # 浮點數 float
            print(123.0, type(123.0))

            # ex(+).
            # 估算結果
            """

            a = 7 + 2 - 3 * 2
            print('a:', a)
            b = 8 + 4 / 2 - 5 * 2
            print('b:', b)
            
        
            
            """
            # ex(+).
            # 估算結果
            """

            # 指乘除餘加減
            a = 8 * 2 + 7 - 9 // 4 * 3 ** 2
            print('a:', a)
            b = 27 % 5 * 6 ** 2 - 12 // 6
            print('b:', b)

程式錯誤種類

            
            """
            # 題意：將 a 和 b 以算術加法合計

            1. 語法錯誤 Syntax Error  : 不可執行
            2. 執行錯誤 Runtime Error : 可以執行，過程出錯
            3. 邏輯錯誤 Logic Error   : 可以執行，不會出錯，結果不符預期
            4. 語意錯誤 Semantic Error: 即邏輯錯誤

            type(data): 回傳資料的類別
            int('987'): 將數字轉成整數 -> integer
            str(12345): 將整數轉成數字 -> string
            """

            a = input('Input a number: ))

            print('type of b:', type(b))
            print(a + b)

等差數列

            print()
