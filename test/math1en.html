Math Logic Puzzle

Breaking Down the Ones and Tens Digits

Tens digit Ones digit
8 7
  • Pick any two-digit integer — for example, 87.
  • Now, come up with a calculation rule that can figure out the ones digit and the tens digit of that number.
  • You can only use one of the four basic operations: addition, subtraction, multiplication, or division.

Hint 1

For a number written in base ten, what is the range of possible digits for each place — ones, tens, hundreds, and so on?

Hint 2

Use one of the four basic operations together with one specific positive integer, and observe whether you can get the ones digit or the tens digit as a result.

For example: If you take 87, can adding a certain positive integer give you 8 or 7?

Hint 3

When using division, the result must always be an integer — no fractions or decimals allowed.
